================================================================================
TROUBLESHOOTING
================================================================================

Camera Not Working:
→ Check System Preferences → Security & Privacy → Camera
→ Grant terminal/Python camera access
→ Try: python3 -c "import cv2; print(cv2.VideoCapture(0).isOpened())"

Dependencies Missing:
→ Re-run: bash install_pc_dependencies.sh
→ Check: pip3 list | grep -E "torch|opencv|ultralytics"

Port 8080 In Use:
→ Kill: lsof -ti:8080 | xargs kill -9
→ Or change port in pc_config.yaml

Model Not Found:
→ Verify: ls -lh ../best.pt
→ Should be 6.25 MB
