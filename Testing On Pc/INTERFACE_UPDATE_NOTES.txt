CUSTOMIZATION
================================================================================

To change expiration threshold:
  1. Edit ../frontend/index.html
  2. Find: const EXPIRATION_DAYS = 5;
  3. Change 5 to desired days
  4. Save and refresh browser

To change sales check interval:
  1. Edit ../frontend/index.html
  2. Find: const SALES_CHECK_INTERVAL = 5000;
  3. Change 5000 to desired milliseconds
  4. Save and refresh browser

To add/remove tracked products:
  1. Edit ../frontend/index.html
  2. Find: const TRACKED_PRODUCTS = [...]
  3. Add/remove product names (lowercase)
  4. Save and refresh browser

  → Check browser console
