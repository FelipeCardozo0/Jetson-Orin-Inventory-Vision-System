================================================================================
QUICK START - Two Simple Commands
================================================================================

This folder allows you to test the Jetson inventory system on your PC/Mac
using either the built-in webcam OR your phone camera via USB.

================================================================================
COMMAND 1: Use PC Built-in Webcam
================================================================================

python3 run_pc_webcam.py

✓ Uses camera index 0 (built-in webcam)
✓ No external devices needed
✓ Best for quick testing

================================================================================
COMMAND 2: Use Phone Camera (iPhone via USB)
================================================================================

python3 run_phone_camera.py

✓ Uses camera index 1 (iPhone via USB, configurable)
✓ Higher quality camera
✓ Requires iPhone connected via USB

Setup:
1. Connect iPhone via USB cable
2. Unlock iPhone
3. Tap "Trust This Computer"
4. Run the command

================================================================================
INSTALLATION (One-Time Setup)
================================================================================

1. Install dependencies:
   bash install_pc_dependencies.sh

2. Run one of the commands above

3. Open browser: http://localhost:8080

4. Press Ctrl+C to stop

================================================================================
TROUBLESHOOTING
================================================================================

PC Webcam Not Working:
  → Check camera permissions: System Preferences → Security & Privacy → Camera
  → Grant Terminal/Python camera access
  → Test: python3 -c "import cv2; print(cv2.VideoCapture(0).isOpened())"

Phone Camera Not Working:
  → Verify iPhone is connected via USB
  → Check iPhone is unlocked and trusted
  → Check System Preferences → Camera → iPhone should appear
  → Try different camera index in phone_config.yaml (1, 2, 3, etc.)

Both scripts will show available cameras automatically when run.

================================================================================
CONFIGURATION FILES
================================================================================

PC Webcam:
  File: pc_config.yaml
  Camera Index: 0

Phone Camera:
  File: phone_config.yaml
  Camera Index: 1 (adjust if needed)

To change camera index, edit the respective config file.
